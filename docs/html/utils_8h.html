Calcula a distância euclidiana entre dois pontos.

A distância euclidiana é definida como a raiz quadrada da soma dos quadrados das diferenças entre as coordenadas correspondentes dos pontos:

Esta função é amplamente usada em algoritmos de aprendizado de máquina, como o K-Nearest Neighbors (KNN).

Parâmetros
aPonteiro para o primeiro ponto.
bPonteiro para o segundo ponto.
dimDimensão dos pontos (número de atributos).
Retorna
Distância euclidiana entre os pontos a e b.
