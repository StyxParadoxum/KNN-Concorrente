Descrição detalhada

Interface pública para uma heap de máximo genérica.

Este módulo implementa uma estrutura de dados do tipo heap de máximo, que organiza elementos de modo que o maior valor (com base em um critério de prioridade) fique sempre na raiz. É utilizada aqui para armazenar pares (distância, id), onde a prioridade é dada pelo valor de dist.

A heap é útil em diversos algoritmos de ordenação parcial, busca de k-vizinhos mais próximos, filas de prioridade, entre outros.

Move um elemento para baixo na heap (heapify-down).

Reorganiza a heap de modo que o maior elemento permaneça na raiz. Este procedimento é utilizado após a remoção do elemento máximo, realocando o último elemento para a raiz e restabelecendo a propriedade da heap.
