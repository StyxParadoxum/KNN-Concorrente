Move um elemento para baixo na heap (heapify-down).

Reorganiza a heap de modo que o maior elemento permaneça na raiz. Este procedimento é utilizado após a remoção do elemento máximo, realocando o último elemento para a raiz e restabelecendo a propriedade da heap.

Parâmetros
hPonteiro para uma heap válida.
iÍndice do elemento a ser rebaixado (geralmente 0 após uma remoção).
